## Reading Order

### For Non-Technical Team Members

1. **Start with:** [User Guide (UK)](USER_GUIDE_UK.md) - Sections 1-6
   - Understand what FCP is and why it matters
   - Learn the basic concepts (zones, capabilities, connectors)

2. **Then:** [User Guide (UK)](USER_GUIDE_UK.md) - Section 4
   - Understand current project status and limitations

### For Technical Team Members

1. **Start with:** [Quick Start](QUICK_START.md)
   - Get FCP running on your machine

2. **Then:** [Configuration Examples](CONFIGURATION_EXAMPLES.md)
   - Set up connectors you need

3. **Keep handy:** [Troubleshooting](TROUBLESHOOTING.md)
   - Reference when issues occur

### For Project Managers

1. **Read:** [User Guide (UK)](USER_GUIDE_UK.md) - Sections 1-4, 11-12
   - Understand capabilities and limitations
   - Understand security model
   - Review FAQ
